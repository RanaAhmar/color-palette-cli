A high-performance terminal tool that generates beautiful, accessible color palettes from a single hex seed. Built for developers and designers who value WCAG compliance and professional design systems.

---

## 🚀 Why Color Palette CLI?

Designing accessible user interfaces shouldn't be a chore. **Color Palette CLI** automates the creation of harmonious color scales while providing instant WCAG contrast scores. Whether you're building a sleek dark mode or a high-contrast dashboard, this tool ensures your design is both stunning and inclusive.

### Key Features:
- **Instant Palette Generation:** Create 9-step color scales from any hex seed.
- **WCAG Accessibility Scores:** Real-time contrast ratio calculations (AA/AAA).
- **LCH Interpolation:** Smooth, perceptually accurate color transitions using LCH mode.
- **Zero-Config CLI:** Just run it and get results.
- **Developer First:** Output colors directly to your terminal for fast integration.

## 🛠️ Installation

```bash
# Clone the repository
git clone https://github.com/AhmarHussain/color-palette-cli.git

# Navigate to the directory
cd color-palette-cli

# Install dependencies
npm install

# Link for global use
npm link
```

## 📖 Usage

Generate a palette by providing a hex code:

```bash
color-palette #3498db
```
